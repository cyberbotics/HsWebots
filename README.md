Webots bindings for Haskell

# Usage

```
#setup WEBOTS_HOME and LD_LIBRARY_PATH
source setenv 

#Build
stack build
```

# Supported Platform

* Webots: R2020a revision 1
* OS: ubuntu/bionic and macos(Mojave)
* Build Tools: stack
